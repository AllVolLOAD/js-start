# JS Learning Project

Этот проект был создан в целях изучения программирования на JavaScript.

## Содержание

### II.js
Реализация классической задачи FizzBuzz, которая выводит числа от 1 до 100 с заменой:
- "Fuzz" для чисел, кратных 3
- "Buzz" для чисел, кратных 5
- "FuzzBuzz" для чисел, кратных и 3, и 5

### III.js
Функция `CheezeBoard` для генерации шахматной доски в консоли. Принимает два параметра:
- `ver` - количество вертикальных рядов (по умолчанию 2)
- `hor` - количество горизонтальных клеток в ряду (по умолчанию 8)

## Запуск

Для запуска любого из файлов используйте Node.js:

```bash
node II.js
```

или

```bash
node III.js
```
